# WasamChaudhry.com

Frontend foundation for [WasamChaudhry.com](https://wasamchaudhry.com/), the official personal brand website of Wasam Chaudhry.

## Project Purpose
This project is a luxury, minimal, cinematic personal brand website focusing on Wasam Chaudhry's brand presence, media credibility, and business ventures.

## Stack Used
- Vite
- Vanilla JavaScript
- HTML5
- Tailwind CSS v4
- **No React, No Backend, No Admin Panel, No CMS, No Database**

## SEO & AIO Readiness
This project has been heavily optimized for search engines (SEO) and artificial intelligence answer engines (AIO).
- Fully semantic HTML structure
- Pre-populated JSON-LD Structured Data (Person and WebSite schema)
- `llms.txt` included for LLM crawlers
- Meta tags for Twitter, Open Graph, and standard SEO fully defined
- `robots.txt` and `sitemap.xml` included

## Performance Goals (Lighthouse)
Targeting the highest possible Lighthouse scores:
- Performance: 95+
- Accessibility: 95+
- Best Practices: 95+
- SEO: 100

## Commands

**Installation:**
```bash
npm install
```

**Development Server:**
```bash
npm run dev
```

**Production Build:**
```bash
npm run build
```

**Preview Production Build:**
```bash
npm run preview
```
